Changelog v3: 
• Native OTA Update Support: Sekarang Ultimatum mendukung pembaruan otomatis! Tombol update

• Smart Big Core Killer (Universal): Algoritma deteksi otomatis untuk Little Core dan mematikan Big/Prime Core. 

• Hardware-Level Underclocking: Opsi Low/Med/High kini jauh lebih presisi.

• Advanced Storage Trim (Fstrim): Tombol Flush Memory kini mengeksekusi fstrim pada partisi /data, /cache, dan /metadata.

• Aggressive Doze (Powersave Mode): Memaksa Android masuk mode Deep Sleep ekstrem hanya dalam waktu 30 detik setelah layar mati, mengabaikan sensor gerak dan lokasi untuk ketahanan baterai maksimal.

• Touch Anti-Delay (Performance Mode): Mode Performance kini menyuntikkan parameter khusus untuk memangkas touch latency, menaikkan pointer speed, dan memaksimalkan hardware UI rendering.
