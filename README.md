# 🚀 Ultimatum - The Ultimate Android Gaming Module

![Version](https://img.shields.io/badge/Version-v4.0.0-blue)
![Platform](https://img.shields.io/badge/Platform-KernelSU%20%7C%20Magisk-success)
![Status](https://img.shields.io/badge/Status-Universal%20Support-orange)

Ultimatum is a powerful, universal performance module designed to push your Android device to its absolute limits. Equipped with a **Native KSU WebUI**, you can easily monitor hardware and inject performance tweaks directly from your screen without typing any terminal commands.
## 📦 Installation
1. Download the latest `Ultimatum.zip` from the **[Releases](../../releases)** tab.
2. Open **KernelSU** or **Magisk** Manager.
3. Go to the Modules tab -> Install from storage.
4. Select the downloaded ZIP file and flash it.
5. Reboot your device.
6. Open KernelSU Next, find Ultimatum, and click the **WebUI** button to start!

# 🇮🇩 Bahasa Indonesia

Ultimatum adalah modul performa universal yang dirancang untuk memaksimalkan potensi HP Android kamu. Dilengkapi dengan **WebUI bawaan KernelSU**, kamu bisa memantau suhu dan mengatur performa semudah mengusap layar.
## 📦 Cara Pasang
1. Download file `Ultimatum.zip` terbaru di menu **[Releases](../../releases)**.
2. Buka aplikasi **KernelSU** atau **Magisk**.
3. Masuk ke menu Modul -> Install dari penyimpanan.
4. Pilih file ZIP yang sudah didownload.
5. Mulai ulang (Reboot) HP kamu.
6. Buka KernelSU Next, cari Ultimatum, dan klik tombol **WebUI** untuk menikmati fiturnya!
